# Findnation
**Prototyp Centralnej Bazy Danych Rzeczy Znalezionych**

FindNation to działający prototyp systemu, którego celem jest centralizacja zgłoszeń rzeczy znalezionych z różnych samorządów. Projekt przedstawia kompletną ścieżkę danych: od wprowadzenia ich przez urzędnika, przez integrację z zewnętrznymi źródłami (BIP/XML), aż po publiczne udostępnianie w formacie czytelnym maszynowo.

Projekt ilustruje, jak mógłby wyglądać jednolity ogólnokrajowy rejestr rzeczy znalezionych zgodny z wymaganiami dane.gov.pl. 
Prosty do użycia dla urzędników i dostępny publicznie w standardowych formatach.

Projekt stanowi gotowy fundament do wdrożenia ogólnokrajowego rejestru zgodnego z wymaganiami dane.gov.pl oraz standardami otwartych danych:

<https://dane.gov.pl/pl/knowledgebase/useful-materials/program-otwierania-danych-na-lata-2021-2027>
